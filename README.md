3D Antigravity Agent Project

Overview

This project implements a **3D antigravity simulation** where an intelligent agent navigates through obstacles and moves a block toward a goal area. The agent learns to make decisions autonomously using reinforcement learning concepts.

Objective

* Develop a **3D environment** with obstacles
* Train an **AI agent** to avoid barriers
* Ensure the agent successfully **pushes the block to the target (green zone)**

Features

* 3D simulation environment
* Obstacle avoidance logic
* Autonomous agent behavior
* Goal-based reward system
* Expandable for multi-agent interaction

Technologies Used

* Python
* Reinforcement Learning concepts
* (Optional) Unity ML-Agents / Simulation tools
